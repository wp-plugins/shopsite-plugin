Official ShopSite plugin. Allows merchants to insert ShopSite products into their blog.

== Description ==

The plugin integrates ShopSite (http://www.shopsite.com) shopping cart functionality with WordPress blogs. After installing the plugin a merchant will be able to select ShopSite products directly from WordPress admin console and add them to her blog. The visitors of that blog will be able to add products to cart right on the blog, after which they are redirected to the merchant's ShopSite store to check out.

Usage:
There will be new button when adding a new post to the blog, with ShopSite's "SS" on it. Clicking it will bring out a product search form where you can select products from your ShopSite's store database to insert into the post.

== Installation ==

1. Upload shopsite.php, search_products.js, oauth.php, editor_plugin.js, ss-20.png, populate_products.js to the '/wp-content/plugins/shopsite/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure the plugin through the 'Settings'->'ShopSite' menu in Wordpress.
